# 自定义识别词助手

MoviePilot 插件，用于可视化管理自定义识别词，支持标签分类、树形整理和原始文本编辑。

## 1.1.0 交互更新

这一版重点把操作逻辑从“改完再找保存按钮”改成了更直接的模式：

- **新增分类 / 子分类：确认后立即保存**
- **新增识别词：添加后立即保存**
- **编辑识别词：弹窗保存后立即生效**
- **删除识别词 / 分类：确认后立即保存**
- **原始文本页：默认自动保存**
- 顶部仍保留 **手动同步 / 立即保存**，作为兜底按钮
- **空分类 / 空子分类也会持久化保存**，不会因为没有词条就丢失

## 使用说明

### 分类管理

#### 创建分类
1. 点击“添加分类”按钮
2. 输入分类名称
3. 确认后立即保存

#### 创建子分类
1. 在分类行点击“文件夹+”图标
2. 输入子分类名称
3. 确认后立即保存

### 添加识别词

#### 方式一：快速添加
1. 点击“快速添加”按钮
2. 选择分类（可选，留空自动分配到“未分类”）
3. 选择识别词类型
4. 填写相应字段
5. 点击“添加并保存”

#### 方式二：直接输入原始数据
```text
# 动漫分类
测试屏蔽词
原词 => 替换词
前定位 <> 后定位 >> EP+1

# 电影分类
电影屏蔽词
电影原词 => 电影替换词
```

原始数据页支持：
- 输入后自动保存
- 输入框失焦时立即补保存
- 一键格式化后自动保存
- 导入文件后自动保存

## 注意事项

1. 修改识别词前仍建议备份原始数据
2. 确保识别词格式符合要求
3. 自动保存失败时，界面右上角会显示失败状态，可用“立即保存/手动同步”重试
