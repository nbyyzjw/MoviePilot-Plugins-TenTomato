# VarietySpecialMapper

综艺特别篇纠偏插件。

## 解决的问题

很多国产综艺在 PT 站命名时，会把 `纯享`、`陪看`、`夜聊`、`惩罚室`、`聚会`、`彩蛋` 之类的扩展内容继续挂在正片季里，结果 Emby 按 TMDB 刮削时对不上集数。

这个插件会在 **识别阶段** 和 **整理完成后、刮削前** 都做纠偏：

- 识别扩展内容类型
- 在 MP 识别弹窗阶段预先改正季号、集号和 TMDB 指向
- 查询 TMDB 特别篇（S0）集列表
- 计算对应的特别篇集号
- 把文件移动到 `Specials/` 目录
- 把文件名改成 `S00E??`
- 让 MoviePilot 后续按正确季集继续刮削

## 0.4.6 的重点变化

这版把“关键词分类名本身”的管理补上了，同时保留原来的 tags/chips 录入体验：

- **关键词词条仍然是 chips 输入**
  - 输入后按回车即可新增
  - 点击 `×` 即可删除
  - 想改词时，删除后重新输入即可
- **通用关键词分类支持直接管理**
  - 可以改分类名称
  - 可以删除某个分类
  - 可以新增新的分类
- 新增分类时，可以顺手把该分类的源文件关键词 / TMDB 关键词一起填好
- 删除分类后，会同步从规则配置里移除对应类型，避免残留无效项
- **新增节目规则** 仍支持一次把节目基础信息和首个季规则填完，再统一保存
- **新增季规则** 仍支持一次把季号、识别词、关键词、手工匹配填完，再统一保存
- `manual_matches` 文本框内保留了 **模板和示例**
- 仍支持 **源文件季号** 和 **TMDB 实际季号** 分离
- 仍支持 **TMDB 季识别关键词** 处理只写年份或固定文案的特别篇

规则优先级：

1. 当前节目的当前季规则
2. 通用关键词库

## 配置结构

现在配置页按下面的树状结构组织：

- 通用关键词库
  - 先导
  - 彩蛋
  - 企划
  - 加更
- 节目规则
  - 节目 A
    - 第 1 季规则（来源季号 / TMDB 正片季号 / TMDB 季识别关键词）
      - 先导
      - 彩蛋
      - 企划
      - 加更
      - 该节目已有的专属类型
    - 第 2 季规则
      - ...

## 使用说明

### 编辑关键词

- 关键词框现在支持 **直接输入后回车新增**
- 点击关键词右侧 `×` 可以删除
- 想修改某个关键词时，**删掉后重新输入** 即可
- 左侧填：**源文件名关键词**
- 右侧填：**TMDB 标题关键词**
- 通用词库现在只保留最基础的 4 类，节目特化请在节目规则里填

### 新增节目

在“新增节目规则”里填写：
- 节目名称
- TMDB ID
- 别名
- 默认主季
- 首个季号
- 首个季的关键词
- 首个季的 TMDB 季识别关键词（可选）
- 首个季的高级手工匹配（可选）
- 额外初始化季号（可选）

最后保存配置。

现在 **不需要额外勾选创建开关**，只要“节目名称 + TMDB ID”有值，保存时就会自动创建。

### 新增季规则

进入对应节目面板，在“新增一个季规则”里填写：
- 新季号
- TMDB 正片季号（可选）
- TMDB 季识别关键词（可选）
- 各类型关键词
- 高级手工匹配（可选）

最后保存配置。

现在 **不需要额外勾选创建开关**，只要“新季号”有值，保存时就会自动创建。

### 季号错位怎么填

如果源文件写的是 `S09`，但 TMDB 实际正片季是 `8`，就在该季规则里这样填：

- `来源季号`: `9`
- `TMDB 正片季号`: `8`
- `TMDB 季识别关键词`: 比如 `2026`

这样遇到 TMDB 特别篇标题像：
- `2026 特别企划`
- `2026 加更版第1期`
- `2026 超前彩蛋第1期`

也能自动归到来源 `S09` 这季去映射。

### 删除节目 / 删除季

直接在对应面板勾选：
- `保存时删除这个节目`
- `保存时删除这一季规则`

然后保存配置。

## 高级说明

- `manual_matches` 仍然保留，用于极少数非常规命名
- 当前 UI 已把 **关键词规则** 做成可交互管理
- `manual_matches` 作为高级兜底项，仍然是 JSON 文本
- 推荐格式：

```json
[
  {
    "type": "bonus",
    "source_keywords": ["超前彩蛋"],
    "index": 1,
    "target_episode": 3
  }
]
```

- 含义：当文件名命中 `超前彩蛋`，并识别到是第 `1` 期时，强制映射到特别篇第 `3` 集
- 当前已内置《喜人奇妙夜》和《妻子的浪漫旅行》的默认示例规则

## 注意

- 当前版本只处理 **本地存储**
- 如果节目命名非常野，建议补 `manual_matches`
- 推荐先靠极简通用词库跑，再给个别节目、个别季补专属关键词
