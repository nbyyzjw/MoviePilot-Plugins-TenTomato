# VarietySpecialMapper

综艺特别篇纠偏插件。

## 解决的问题

很多国产综艺在 PT 站命名时，会把 `纯享`、`陪看`、`夜聊`、`惩罚室`、`聚会`、`彩蛋` 之类的扩展内容继续挂在正片季里，结果 Emby 按 TMDB 刮削时对不上集数。

这个插件会在 **整理完成后、刮削前** 做一次纠偏：

- 识别扩展内容类型
- 查询 TMDB 特别篇（S0）集列表
- 计算对应的特别篇集号
- 把文件移动到 `Specials/` 目录
- 把文件名改成 `S00E??`
- 让 MoviePilot 后续按正确季集继续刮削

## 0.3.0 的重点变化

这版不再要求你直接维护一大段 JSON，而是改成了结构化的交互式配置：

- **通用关键词库**
- **节目规则**
- **每部综艺按季拆分**
- **每季下再按类型管理关键词**
- 支持在配置页里直接 **新增节目、删除节目、新增季、删除季、编辑关键词**

规则优先级现在是：

1. 当前节目的当前季规则
2. 通用关键词库

也就是说：
- 同一部综艺不同季可以用不同关键词
- 某一季如果定义了自己的 `watch / bonus / party ...` 关键词，会优先覆盖通用词库

## 配置结构

现在配置页按下面的树状结构组织：

- 通用关键词库
  - pilot
  - pure
  - watch
  - chat
  - punish
  - party
  - bonus
- 节目规则
  - 节目 A
    - 第 1 季规则
      - pilot
      - pure
      - ...
    - 第 2 季规则
      - ...
  - 节目 B
    - 第 1 季规则
      - ...

## 通用类型示例

- `pilot`: 先导、先导片、抢先看、开放日、集结篇
- `pure`: 纯享、纯享版
- `watch`: 陪看、专门陪你看
- `chat`: 夜聊、聊天局、唠嗑
- `punish`: 惩罚室、不好笑惩罚室
- `party`: 聚会、派对、狼人杀、游戏时间、课间游戏
- `bonus`: 超前彩蛋、彩蛋、加更、企划、特别企划、花絮、特辑、高光、回顾、团建

## 使用说明

### 编辑关键词

- 每个关键词输入框建议 **每行一个**
- 左侧填：**源文件名关键词**
- 右侧填：**TMDB 标题关键词**

### 新增节目

在“新增节目规则”里填写：
- 节目名称
- TMDB ID
- 别名
- 初始化季号

然后勾选：
- `保存时新增这个节目`

最后保存配置。

### 新增季规则

进入对应节目面板，在“新增一个季规则”里填写：
- 新季号

然后勾选：
- `保存时新增该季`

最后保存配置。

### 删除节目 / 删除季

直接在对应面板勾选：
- `保存时删除这个节目`
- `保存时删除这一季规则`

然后保存配置。

## 高级说明

- `manual_matches` 仍然保留，用于极少数非常规命名
- 当前 UI 主要把 **关键词规则** 做成可交互管理
- `manual_matches` 作为高级兜底项，仍然是 JSON 文本

## 注意

- 当前版本只处理 **本地存储**
- 如果节目命名非常野，建议补 `manual_matches`
- 推荐先靠通用词库跑，再给个别节目、个别季补专属关键词
