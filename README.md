# MoviePilot-Plugins-TenTomato

老大的 MoviePilot 插件仓库。

## 当前插件

- **SimpleTextBoard / 简单文本面板**
  - 一个最小可用的 MoviePilot V2 示例插件
  - 支持配置标题、正文
  - 支持在插件详情页和仪表板展示文本

## 仓库结构

```text
.
├── icons/
├── package.v2.json
└── plugins.v2/
    └── simpletextboard/
        ├── __init__.py
        └── README.md
```

## 在 MoviePilot 中添加插件市场

将这个仓库地址加入插件市场：

```text
https://github.com/nbyyzjw/MoviePilot-Plugins-TenTomato
```

然后安装 `SimpleTextBoard` 即可。
